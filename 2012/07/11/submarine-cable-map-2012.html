Submarine Cable Map 2012

July 11, 2012

TeleGeography has released an interactive world map of all submarine internet cables with different data about “Landing Stations”, “Pricing Trends”, “Construction Cost”, “Used International Bandwith”, etc.

You can check out the map here.

If you want to buy the map you can do that for the price of $250.
